Example Usage of the Scripting Console

Example 1: hide all ensemble memberships
For JRE7, use: 
	var it = memberships.iterator()
	while (it.hasNext()){
		var el = it.next()
		el.setVisible(false)
	}
When using JRE 8, there is a simpler alternative for iterating the collection:
	for each (var el in memberships){
		el.setVisible(false)
	}
Example 2: show only ensemble memberships with coordinator V7
For JRE7, use: 
	var it = memberships.iterator()
	while (it.hasNext()){
		var el = it.next()
		if (el.getCoordinator().equals("V7")){
			el.setVisible(true)
		} else {
			el.setVisible(false)		
		}
	}
And for JRE 8 we can use:
	for each (var el in memberships){
		if (el.getCoordinator().equals("V7")){
			el.setVisible(true)
		} else {
			el.setVisible(false)		
		}
	}
Example 3: print info about ensemble memberships.
For JRE7, use: 
	var count = 0;
	var it = memberships.iterator()
	while(it.hasNext()){
		var el = it.next()
		println("Item " + count + ": ")
		println("Ensemble name=" + el.getEnsembleName() + ", ")
		println("Coordinator=" + el.getCoordinator() + ", ")
		println("Member=" + el.getMember())
		println("")
		count += 1
	}
For JRE8, use:
	var count = 0;
	for each(var el in memberships){
		print("Item " + count + ": ")
		print("Ensemble name=" + el.getEnsembleName() + ", ")
		print("Coordinator=" + el.getCoordinator() + ", ")
		print("Member=" + el.getMember())
		print()
		count += 1
	}
Example 4: set image representing all people
	general.setPersonImage("C:/image.png")
Example 5: set image representing persons V7, V11
	general.setPersonImage("C:/image.png", "V7", "V11")
Example 6: set image representing persons V7, V11 - variant II
	var ecmaArray = ["V7","V11"]
	var javaArray = java.lang.reflect.Array.newInstance(java.lang.String, ecmaArray.length)
	for(var i = 0; i<ecmaArray.length; i++){
		javaArray[i] = ecmaArray[i]
	}
	general.setPersonImage("C:/image.png", javaArray)
And for JRE 8 we have
	var ecmaArray = ["V7","V11"]
	var stringArrayType = Java.type("java.lang.String[]")
	var javaArray = Java.to(ecmaArray, stringArrayType)
	general.setPersonImage("C:/image.png", javaArray)
Example 7: set image representing persons V7,V11 back to the original circles
	general.setPersonImage(null, "V7", "V11")
Example 8: set color of the ensemble memberships with coordinator V7 to red, others to green
For JRE7, use: 
	var it = memberships.iterator()
	while(it.hasNext()){
		var el = it.next()
		if (el.getCoordinator().equals("V7")){
			el.setColor(Color.RED)
		} else {
			el.setColor(Color.GREEN)
		}
	}
For JRE8, use:
	for each (var el in memberships){
		if (el.getCoordinator().equals("V7")){
			el.setColor(Color.RED)
		} else {
			el.setColor(Color.GREEN)
		}
	}
Example 9: print the x-coordinates of all the nodes
For JRE7, use: 
	var it = nodes.keySet().iterator()
	while(it.hasNext()){
		var id = it.next()
		println("id=" + id + ", X=" + nodes.get(id).getX())
	}
For JRE8, use:
	for each (var id in nodes.keySet()){
		print("id=" + id + ", X=" + nodes.get(id).getX())
	}
